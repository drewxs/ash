# Ash

WIP; General-purpose procedural programming language.

```go
Lexer -> Parser -> Compiler -> Virtual Machine
|------ Compile Time -----|    |-- Run Time -|

String -> Tokens -> AST -> Bytecode -> Objects
```

-   Interpreter: Tree-walking
-   Compiler: Stack-based VM
